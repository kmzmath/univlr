# UNIVLR Assets

Preencha estas pastas com os arquivos visuais usados nas páginas do UNIVLR.

- `team-logos`: logos de equipes, preferencialmente SVG ou PNG.
- `player-photos`: fotos de jogadores.
- `tournament-icons`: logos/ícones de campeonatos.
- `agent-icons`: ícones de agentes.
- `flags`: bandeiras de estados/países.
- `social-icons`: ícones customizados de redes, caso queira substituir os ícones internos do app.

Os caminhos devem ser referenciados em `team-profiles.json`.
